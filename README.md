# AI Tester Blueprint 3X

A comprehensive learning resource and testing framework for AI and Large Language Models.

## Project Structure

### Chapter 01: LLM Basics
Introduction to Large Language Models and foundational concepts.

- **attention_interactive.html** - Interactive visualization of attention mechanisms
- **attention_is_all_you_need.html** - Reference material based on the "Attention Is All You Need" paper

### Chapter 02: Prompt Engineering
Techniques and strategies for effective prompt engineering with LLMs.

- **Notes.md** - Study notes and key learnings
- **prompting_notes.md** - Detailed prompting techniques and best practices
- **RICE_POT.md** - Framework documentation for structured prompting

## Getting Started

1. Navigate to the relevant chapter directory
2. Review the markdown notes for theoretical understanding
3. Open HTML files in a browser for interactive visualizations
4. Apply techniques and test with your LLM of choice

## Contents Overview

This blueprint covers:
- Fundamental concepts of Large Language Models
- Attention mechanisms and transformers
- Effective prompt engineering strategies
- Practical frameworks for structured prompting (RICE_POT)

## Usage

Each chapter builds upon the previous one. Start with Chapter 01 for foundational knowledge, then move to Chapter 02 for practical application of prompt engineering techniques.
